## 你好，世界！

欢迎来到我的个人博客！这是我在这个小站上写下的第一篇文章。

### 为什么要建这个博客？

作为一名前端开发者，我一直想拥有一个属于自己的空间，用来记录学习笔记、技术心得，以及生活中的一些思考。经过一番调研，我最终选择了 **Astro 6** 作为博客的技术栈。

Astro 的理念非常打动我——默认零 JavaScript，只在需要时才加载交互组件。这对于一个以内容为主的博客来说，简直是完美的选择。

### 技术栈一览

这个博客使用了以下技术：

```typescript
const techStack = {
  framework: 'Astro 6',
  ui: 'React 19',
  styling: 'Tailwind CSS 4',
  components: 'shadcn/ui',
  language: 'TypeScript',
  deployment: 'Vercel',
};

console.log('博客技术栈:', techStack);
```
